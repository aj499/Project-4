How to use ______

  1. Start the program and then select between the economic or health checkboxes. That will select which world map will appear on the next screen.
  2. Selecting the health map checkbox will bring up a color-coded world map of estimated life expectancy at birth, circa 2008. Selecting the economic view will bring up a color-coded map of GDP per capita.
  3. Then enter your name in the text box and then click the Start button.
  4. Then using the mouse select the button on the continent you would like to explore.
  5. While on the specific continent view, you can click the “change mode” button to toggle between quiz mode and explore mode.
  6. In explore mode you will first see the various data for the current continent.
  7. Then select a country that has a button on it and it will bring up the data for that country.
